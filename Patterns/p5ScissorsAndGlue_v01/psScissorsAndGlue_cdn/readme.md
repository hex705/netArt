## Scissors and Glue for p5js

This project provides a simple way to construct String based delimited message for sending over serial and MQTT and other comm channels that support Strings (eg. sockets).

The stucture of the messages is to facilite parsing in arduino microcontrollers.

This library pairs with arduino libraries scissors and glue.

Together these allow for easy assembly and parsing of simple delimited string based messages.

### Get Scissors and Glue for Arduino:

Both libraries are in the Arduino library manager.  Search for *scissors* OR *glue* OR *hex705* (author)
